For this task, I used HackerRank for coding practice and Google 
Forms for collaboration tools. On HackerRank, I completed a 
beginner-level challenge, which helped me understand how coding 
platforms improve logical thinking and problem-solving. It also 
introduced me to the importance of regular practice and skill-based 
learning. 
For the collaboration part, I created a Digital Literacy Awareness Quiz 
using Google Forms with five questions, including multiple-choice and 
short-answer types. I linked the responses to Google Sheets to 
observe how data collection and response management work in real 
time. This task showed me how digital collaboration tools can be used 
not only for assignments but also for surveys, feedback, and academic 
teamwork. 
The most valuable learning from this task was that digital platforms 
can make academic work more interactive and efficient. However, 
their effectiveness depends on how responsibly students use them.
