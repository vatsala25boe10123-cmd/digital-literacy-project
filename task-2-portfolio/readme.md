In this task, I created or updated my profiles on GitHub, LinkedIn, and 
Kaggle. I chose these three platforms because each serves a different 
professional purpose. GitHub is useful for storing and showcasing 
coding projects, LinkedIn helps build a professional network and 
academic identity, and Kaggle introduces students to data science and 
practical problem-solving. 
On GitHub, I created a profile README that includes my name, 
branch, year, and my learning goals. On LinkedIn, I completed my 
education details with my current degree and university. Kaggle gave 
me exposure to how students can participate in beginner-level 
datasets and challenges. Over the next four years, I plan to use 
GitHub to document projects, LinkedIn to build my professional 
network, and Kaggle to strengthen my analytical and technical skills. 
This task taught me that digital presence should focus on authenticity 
rather than exaggeration. A simple and genuine profile can create 
better long-term opportunities than a profile built only for appearance.
